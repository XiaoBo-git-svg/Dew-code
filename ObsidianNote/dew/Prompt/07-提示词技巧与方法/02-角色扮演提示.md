# 角色扮演提示

> 给 AI 设定一个角色，它的回答会更专业、更符合场景。这是最简单但最有效的技巧之一。

---

## Prompt 模板

```
你是一名 [专业身份]，拥有 [年限/经验] 的 [领域] 经验，擅长 [具体技能/风格]。

[任务描述]：[你想让 AI 做什么]

要求：
1. [具体要求1]
2. [具体要求2]

[待处理的内容]：
[你的内容]
```

---

## 使用示例

```
你是一名经验丰富的 Python 开发工程师，拥有 8 年后端开发经验，擅长代码审查和性能优化。

请帮我看一下这段代码，找出潜在的性能问题和安全隐患。

要求：
1. 逐行分析关键逻辑
2. 给出具体的优化建议
3. 用代码示例说明改进方案

待处理的内容：
def get_users(db):
    users = db.query("SELECT * FROM users")
    result = []
    for user in users:
        orders = db.query(f"SELECT * FROM orders WHERE user_id = {user['id']}")
        user['orders'] = orders
        result.append(user)
    return result
```

---

## 角色设定越具体，效果越好

| ❌ 太笼统 | ✅ 足够具体 |
|-----------|------------|
| "你是个专家" | "你是有 5 年电商数据分析经验的专家，特别擅长用户行为分析" |
| "你是程序员" | "你是有 10 年经验的 Java 架构师，擅长微服务设计和性能调优" |
| "你是老师" | "你是高中数学特级教师，善于用生活中的例子解释抽象概念" |

## 角色扮演进阶用法

### 多角色讨论

```
请分别以产品经理、设计师、开发工程师三个角色，讨论"是否应该给 App 加深色模式"。
每个角色给出自己的立场和理由，最后给出一个综合建议。
```

### 角色 + 场景

```
你是一名急诊科医生，现在是凌晨 3 点，一名患者描述了以下症状...
请以医生的专业判断，给出初步诊断方向和建议。
```

### 苏格拉底式引导

```
你是一位苏格拉底式的老师，不要直接告诉我答案。
通过提问引导我自己思考和发现答案。如果我答对了，给我更深的问题；
如果我答错了，给我提示而不是答案。
```

---

## 技巧

- **角色要具体**：加上年限、擅长领域、风格特征
- **配合任务一起说**：角色 + 任务 + 格式，三者结合效果最佳
- **不同角色问同一个问题**：获得多元视角
- **角色不是万能的**：AI 本质还是在做文本生成，专业问题仍需自己验证
