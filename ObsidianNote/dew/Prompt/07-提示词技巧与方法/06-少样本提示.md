# 少样本提示

> 给 AI 几个高质量示例，让它理解你想要的输入输出模式——3-5 个示例通常就够。

---

## Prompt 模板

```
[任务说明]

【示例 1】
输入：[示例输入 1]
输出：[示例输出 1]

【示例 2】
输入：[示例输入 2]
输出：[示例输出 2]

【示例 3】
输入：[示例输入 3]
输出：[示例输出 3]

【你的任务】
输入：[实际输入]
输出：
```

---

## 使用示例

```
请分析以下文本的情感，只返回"正面"、"负面"或"中性"，不要解释。

【示例 1】
输入：这个教程写得太好了，通俗易懂！
输出：正面

【示例 2】
输入：这个功能太难用了，浪费我很多时间。
输出：负面

【示例 3】
输入：今天天气一般，不冷也不热。
输出：中性

【你的任务】
输入：这个 Prompt 教程很实用，但有些地方可以更详细一点。
输出：
→ AI 输出：中性
```

---

## 技巧

- **质量优于数量**：3-5 个高质量示例比 10 个低质量示例更有效
- **覆盖多样性**：示例应覆盖不同情况（如正面/负面/中性），不要全是同一类
- **格式统一**：每个示例的输入输出格式必须一致，AI 会模仿你的格式
- **明确边界**：用"只返回 XXX，不要解释"等指令限制输出范围
- **三种模式**：零样本（直接提问）、单样本（给 1 个例子）、多样本（给 2-5 个例子），根据任务复杂度选择
