# 思维链提示

> 让 AI 把思考过程说出来，而不是直接给答案——通过「一步步思考」大幅提升复杂问题的正确率。

---

## Prompt 模板

```
[你的问题]

让我们一步步思考：
1. 先明确题目给出了哪些已知条件
2. 再明确需要求解什么
3. 逐步推导，每一步都写清楚
4. 最后给出最终答案
```

---

## 使用示例

```
一个池塘里的睡莲，每天面积都会翻倍。如果 48 天能长满整个池塘，那么长满半个池塘需要多少天？让我们一步步思考。

→ AI 思维过程：
1. 题目说睡莲每天面积翻倍
2. 第 48 天长满整个池塘
3. 往前倒推一天，第 47 天面积是第 48 天的一半
4. 所以第 47 天长满半个池塘

答案：47 天
```

---

## 技巧

- **触发关键词**：在问题末尾加"让我们一步步思考"或"Let's think step by step"即可激活思维链
- **适用场景**：数学题、逻辑推理、代码调试、需要多步分析的问题
- **不适用场景**：简单问题（如"今天星期几"）不需要思维链，反而浪费 token
- **结合编号步骤**：要求 AI 按 1、2、3 分步写出推理过程，比笼统的"想想"更有效
- **先列条件再推导**：让 AI 先整理已知信息，再开始推导，减少遗漏
