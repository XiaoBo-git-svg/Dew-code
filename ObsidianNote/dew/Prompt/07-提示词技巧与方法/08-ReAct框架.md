# ReAct 框架

> ReAct = Reasoning（推理）+ Acting（行动）——让 AI 先思考做什么，再执行，然后观察结果，循环直到任务完成。

---

## Prompt 模板

```
请用以下步骤解决问题：

【思考】分析当前状态，决定下一步做什么
【行动】执行具体操作（如搜索、计算、查询）
【观察】记录行动结果
（重复 思考→行动→观察 直到得出最终答案）
【完成】给出最终答案

问题：[你的问题]
```

---

## 使用示例

```
问题：北京和深圳哪个城市人口更多？

🤔 思考：要比较两地人口，我需要先查数据
⚡ 行动：搜索 → 北京人口
👁️ 观察：北京约 2189 万

🤔 思考：现在查深圳的数据
⚡ 行动：搜索 → 深圳人口
👁️ 观察：深圳约 1768 万

🤔 思考：2189 > 1768，可以得出结论
✅ 完成：北京的人口比深圳多
```

---

## 技巧

- **适用场景**：需要查询外部信息、多步骤任务、需要验证结果、工具调用（计算器、API 等）
- **不适用场景**：简单问题（如"1+1 等于几"）不需要 ReAct，直接回答即可
- **格式清晰**：用【思考】【行动】【观察】标签分隔每一步，便于追踪推理过程
- **工具组合**：ReAct 天然适合与外部工具（搜索、数据库、代码执行）结合使用
- **循环控制**：提示 AI "重复直到任务完成"，避免过早结束或无限循环
