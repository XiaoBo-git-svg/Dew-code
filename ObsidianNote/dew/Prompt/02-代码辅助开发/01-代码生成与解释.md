# 代码生成与解释

> 写重复性代码、查文档、写样板代码——这些事情 AI 最擅长。

---

## Prompt 模板

### 写样板代码

```
请帮我用 [语言] 写一个 [功能描述]，要求：
1. 有注释
2. 处理异常
3. [其他要求]

功能细节：
[具体需求描述]
```

### 技术选型对比

```
我要做一个 [项目类型]，在 [技术 A] 和 [技术 B] 之间纠结。

请从以下维度帮我对比：
1. 性能
2. 学习曲线
3. 生态（社区、文档、第三方库）
4. 适用场景

最后给我一个推荐。
```

### 代码解释

```
请帮我解释这段代码在做什么：
1. 逐行说明
2. 最后给一个整体总结
3. 指出可能的改进点

代码：
[你的代码]
```

### 添加注释

```
请帮我给这段代码添加合适的注释，重点说明：
1. 设计思路
2. 关键逻辑
3. 不要给每一行都加无意义的注释

代码：
[你的代码]
```

---

## 使用示例

```
请帮我用 Python 写一个读取 CSV 文件并统计每列数据类型的函数，要求：
1. 有注释
2. 处理异常
3. 返回一个字典，key 是列名，value 是数据类型统计

功能细节：
- 输入：CSV 文件路径
- 输出：每列的数据类型（int、float、str、date 等）及占比
- 如果某列有混合类型，标注出来
```

---

## 技巧

- **给出完整上下文**：告诉 AI 你用的语言、框架、版本
- **指定代码风格**：比如"遵循 PEP 8"或"用函数式风格"
- **要求有注释**：这样你更容易理解和修改
