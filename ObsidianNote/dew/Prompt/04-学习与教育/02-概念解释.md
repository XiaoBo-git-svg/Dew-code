# 概念解释

> 遇到不懂的概念，让 AI 用多种方式解释给你听。

---

## Prompt 模板

```
请用几种不同的方式解释 [概念]。

要求：
1. 用大白话解释（不要用术语）
2. 用一个生活中的例子类比
3. 用一个简单的代码/数学例子演示
4. 解释它和 [相似概念] 的区别
5. 指出初学者最容易误解的地方

我目前的理解是：[你的理解，可写可不写]
```

---

## 使用示例

```
请用几种不同的方式解释"什么是机器学习"。

要求：
1. 用大白话解释
2. 用生活中的例子类比
3. 和"传统编程"做对比
4. 指出初学者最容易误解的地方
```

---

## 费曼学习法

> 如果你不能用简单的语言解释一个概念，说明你还没有真正理解它。

```
请用费曼学习法帮我理解 [概念]。

要求：
1. 假设我是 10 岁小孩，用最简单的语言解释
2. 用一个日常生活的例子
3. 避免使用专业术语
4. 如果必须用术语，先解释术语本身
```

---

## 技巧

- **说出你的理解**：告诉 AI 你目前的理解，它能针对性地纠正
- **要求对比**：让 AI 对比相似概念，避免混淆
- **要求反例**：知道"什么不是"和知道"什么是什么"同样重要
- **多问几次**：第一次没理解，换个角度再问
