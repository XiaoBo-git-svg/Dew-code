# 语言学习

> AI 是练习外语的好伙伴——它能陪你对话、帮你改作文、解释语法。

---

## Prompt 模板

### 作文修改

```
请帮我修改这篇 [语言] 作文。

要求：
1. 指出语法错误
2. 指出用词不当的地方
3. 给一个更地道的版本
4. 解释为什么这样改

作文：
[你的作文]
```

### 对话练习

```
我想练习 [语言] 的口语，主题是 [主题]。

规则：
1. 请扮演 [角色]（比如：咖啡店店员、机场工作人员、朋友）
2. 每次回复不要超过 3 句话
3. 用简单的词汇
4. 如果我有语法错误，请用括号指出来
5. 如果我说得太复杂，给我一个更简单的说法

开始吧！
```

### 词汇学习

```
我在学习 [语言]，请帮我用以下单词/短语造句。

单词：[单词列表]

要求：
1. 每个词造 5 个句子
2. 要日常场景中真的会用的
3. 难度递进：从简单到复杂
4. 标注音标/重音（如果适用）
```

### 文化背景

```
我要去 [国家/地区]，请给我介绍一些当地的文化习俗和注意事项。

重点关注：
1. 日常礼仪（打招呼、用餐、送礼）
2. 容易冒犯到别人的行为
3. 常用的礼貌用语
4. 小费/礼物的文化
```

## 使用示例

```
我想练习英语的口语，主题是在咖啡店点单。

规则：
1. 请扮演咖啡店店员
2. 每次回复不要超过 3 句话
3. 用简单的词汇
4. 如果我有语法错误，请用括号指出来
5. 如果我说得太复杂，给我一个更简单的说法

开始吧！
```

---

## 技巧

- **角色扮演**：让 AI 扮演特定角色，练习更真实
- **要求纠错**：明确告诉 AI 要指出你的错误
- **从简单开始**：先用简单词汇对话，逐渐增加难度
- **每天坚持**：10 分钟对话练习比一周一次长时间练习更有效
