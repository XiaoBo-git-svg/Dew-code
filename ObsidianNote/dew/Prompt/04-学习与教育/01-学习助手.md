# 学习助手

> 最好的学习方式不是"看视频"，而是"对话"——有问题马上问，有理解偏差马上纠正。

---

## Prompt 模板

```
我想学习 [主题]，我目前的水平是 [水平描述]。

请帮我制定一个学习计划：
1. 先评估一下我应该从哪里开始
2. 给一个 4 周的学习路径，每周有明确的目标
3. 推荐一些学习资源（不需要具体链接，说类型就行）
4. 每周给一个小练习，检验学习效果

我的背景：
- 相关经验：[比如：零基础 / 懂一点 Python / 用过类似工具]
- 学习目标：[比如：能做一个简单的网站 / 通过某个考试 / 工作需要]
- 每周学习时间：[比如：5 小时 / 10 小时]

请用苏格拉底式教学法——多问我问题，引导我思考，而不是直接给答案。
```

---

## 苏格拉底式教学法

> 让 AI 多问你问题，引导你思考，而不是直接给答案。

这样学得更深入，记忆更持久。

```
请用苏格拉底式教学法教我 [主题]。

规则：
1. 不要直接告诉我答案
2. 通过提问引导我思考
3. 如果我答对了，给我更深的问题
4. 如果我答错了，给我提示而不是答案
5. 每次只问一个问题
```

---

## 使用示例

```
我想学习 Python 数据分析，我目前的水平是会基础的 Python 语法，写过几个小脚本。

请帮我制定一个学习计划：
1. 先评估一下我应该从哪里开始
2. 给一个 4 周的学习路径，每周有明确的目标
3. 推荐一些学习资源（不需要具体链接，说类型就行）
4. 每周给一个小练习，检验学习效果

我的背景：
- 相关经验：会基础 Python 语法，用过列表和字典，没接触过数据分析库
- 学习目标：能用 Python 处理 Excel 数据，生成图表，做简单的数据分析报告
- 每周学习时间：8 小时

请用苏格拉底式教学法——多问我问题，引导我思考，而不是直接给答案。
```

---

## 技巧

- **诚实描述水平**：告诉 AI 你的真实水平，它才能给出合适的学习路径
- **设定明确目标**："能做一个网站"比"学编程"更有效
- **要求练习题**：光看不练学不会
- **随时追问**：不懂就问，AI 不会嫌你烦
