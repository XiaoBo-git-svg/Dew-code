# 简历优化

> 好简历不是堆砌经历，而是用精准的语言展示你的价值。AI 能帮你把"做了什么"变成"做到了什么"。

---

## Prompt 模板

```
你是一名资深的 HR 顾问，请帮我优化以下简历描述。

优化要求：
1. 用"动词 + 成果"的方式改写
2. 量化工作成果（用数据说话）
3. 使用行业关键词（方便 ATS 系统筛选）
4. 突出个人贡献而非团队成果

原始描述：
[你的简历内容]

目标岗位：[你应聘的职位]
行业：[所在行业]
```

---

## 使用示例

### 技术岗位优化

```
你是一名资深的 HR 顾问，请帮我优化以下简历描述。

优化要求：
1. 用"动词 + 成果"的方式改写
2. 量化工作成果
3. 使用技术岗位常见关键词
4. 突出个人贡献

原始描述：
负责公司后台系统的开发和维护，参与了多个项目，写了接口和前端页面，修复了一些 Bug。

目标岗位：高级后端开发工程师
行业：互联网
```

**优化后示例：**
- 独立设计并开发 3 个核心微服务模块，日均处理 50 万+ API 请求，P99 响应时间 < 200ms
- 主导数据库查询优化，将关键接口响应时间从 2s 降低至 300ms，提升系统吞吐量 6 倍
- 建立自动化测试体系，代码覆盖率从 30% 提升至 85%，线上 Bug 率下降 40%

## 简历优化要素

| 要素 | 优化前（❌） | 优化后（✅） |
|------|-------------|-------------|
| 动词开头 | "负责用户增长" | "主导用户增长策略，实现月活提升 35%" |
| 量化成果 | "提升了系统性能" | "优化数据库查询，响应时间降低 70%" |
| 个人贡献 | "参与了项目" | "独立负责支付模块开发，覆盖 3 个支付渠道" |
| 关键词 | "写了接口" | "设计 RESTful API，支持 10 万+ 并发调用" |

## 不同场景的简历模板

### 应届生/实习生

```
你是一名校招 HR 顾问。我是 [专业] 的应届生，应聘 [岗位]。

以下是我的经历（可能不太丰富），请帮我：
1. 挖掘课程项目、社团经历、实习中的亮点
2. 用专业语言描述，但不要夸大
3. 突出学习能力和成长潜力
4. 针对目标岗位调整关键词

我的经历：
[你的描述]
```

### 转行求职

```
你是一名擅长职业转型的 HR 顾问。我想从 [原行业/岗位] 转到 [目标行业/岗位]。

请帮我：
1. 找出可迁移的技能和经验
2. 重新包装经历，突出与目标岗位的关联
3. 弱化不相关的技术细节
4. 建议我补充哪些方面

原始简历：
[你的简历]
```

---

## 技巧

- **一次只优化一段**：不要把整份简历丢给 AI，逐段优化效果更好
- **给出目标岗位**：不同岗位需要突出不同的能力
- **保持真实**：AI 可以帮你美化表达，但不要编造经历
- **多版本对比**：让 AI 给出 2-3 个优化版本，选最好的
