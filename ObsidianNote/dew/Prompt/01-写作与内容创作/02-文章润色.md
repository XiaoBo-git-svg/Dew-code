# 文章润色

> AI 生成的内容只是初稿，你需要根据实际情况调整细节。但至少，你不用从零开始写了。

---

## Prompt 模板

```
请帮我润色以下文字，保持核心意思不变，但让它更 [风格]。

风格选项：更简洁、更专业、更口语化、更有感染力、更有逻辑

要求：
1. 保持原意，不要添加新信息
2. 让句子更通顺流畅
3. 去除冗余表达

需要润色的内容：
[你的文字]
```

---

## 使用示例

```
请帮我润色以下文字，保持核心意思不变，但让它更简洁专业。

要求：
1. 保持原意，不要添加新信息
2. 让句子更通顺流畅
3. 去除冗余表达

需要润色的内容：
我们公司最近在做一个新的项目，这个项目的主要目的是为了提高用户的体验感，让用户在使用我们的产品的时候能够感觉到更加的舒适和方便，所以说我们需要对产品进行一些改进和优化。
```

---

## 技巧

- **明确风格方向**：告诉 AI 你想要"更简洁"还是"更专业"，效果差异很大
- **保持原意**：强调"不要添加新信息"，防止 AI 过度发挥
- **分段润色**：长文章可以分段处理，效果更好
