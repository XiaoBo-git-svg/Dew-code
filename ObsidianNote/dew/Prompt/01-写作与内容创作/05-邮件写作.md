# 邮件写作

> 职场人每天写邮件、写报告，这是最能通过 AI 省时间的场景。

---

## Prompt 模板

```
请帮我写一封 [邮件类型]。

收件人：[收件人身份，如：领导、客户、团队成员]
核心信息：[你想说什么，用 bullet point 列出]
期望行动：[希望对方做什么]
语气要求：[正式/友好/简洁/详细]

参考信息：
[背景信息，如：之前的对话、相关数据]
```

---

## 使用示例

### 会议改期通知

```
请帮我写一封邮件。

收件人：项目组全体成员
核心信息：
- 原定于周五的项目评审会改到下周三下午 2 点
- 会议室不变，还是 302
- 请大家准备好 5 分钟的进度汇报
- 有特殊情况不能参加的，明天中午前告诉我

期望行动：收到请回复，确认能否参加
语气要求：正式但友好，不要太生硬

参考信息：这是项目的第 3 次评审会
```

---

## 邮件类型速查

| 邮件类型 | 关键要素 |
|----------|----------|
| 会议通知 | 时间、地点、议题、准备事项 |
| 进度汇报 | 完成情况、问题、下一步计划 |
| 客户回复 | 问题确认、解决方案、时间节点 |
| 申请审批 | 申请内容、理由、预期结果 |
| 催办提醒 | 事项、截止时间、影响说明 |

---

## 技巧

- **Bullet point 列信息**：比写成段落更清晰，AI 理解更准确
- **明确语气**：同一内容，"正式"和"友好"差别很大
- **给出期望行动**：告诉 AI 你希望收件人做什么
