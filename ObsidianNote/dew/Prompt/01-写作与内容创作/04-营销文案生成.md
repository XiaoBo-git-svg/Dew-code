# 营销文案生成

> 写文案是 AI 的强项——它能快速生成多个版本，帮你找到最佳表达。

---

## Prompt 模板

### 产品卖点提炼

```
我有一个 [产品类型]，主要功能是 [功能列表]，目标用户是 [用户画像]。

请帮我提炼 3 个核心卖点，每个卖点用一句话表达。
```

### 朋友圈文案

```
我想在朋友圈推广 [产品/服务]，请帮我写 5 个不同风格的文案版本，每个不超过 100 字，要加合适的表情符号。
```

### 邮件营销标题

```
我要写一封推广邮件，主题是 [邮件主题]，请帮我想 10 个可能的标题，分为"直接型"、"好奇型"、"优惠型"三类。
```

### 落地页文案

```
请帮我为 [产品] 写一个 landing page 的文案结构，包含：
1. 标题
2. 副标题
3. 3 个核心价值
4. 常见问题解答
```

---

## 使用示例

```
我有一个智能保温杯，主要功能是实时显示水温、提醒喝水、手机 APP 记录每日饮水量，目标用户是注重健康的上班族。

请帮我提炼 3 个核心卖点，每个卖点用一句话表达。
```

---

## 技巧

- **多版本对比**：让 AI 生成多个版本，从中挑选最佳
- **给参考案例**：如果有喜欢的文案风格，直接给 AI 看
- **迭代优化**：第一版不满意，告诉 AI 哪里需要调整
