# 头脑风暴

> 创意的天敌是"大脑一片空白"——AI 能帮你快速生成几十个想法，然后你从中挑几个深入。

---

## Prompt 模板

```
我想为 [目标] brainstorm 一些想法。

背景信息：
- 我们的目标是：[比如：想一个新 slogan / 设计一个线上活动 / 想几个视频选题]
- 目标受众：[谁]
- 风格偏好：[比如：有趣 / 专业 / 感人 / 有冲击力]
- 已有想法：[你已经想到的，可写可不写]
- 不要：[你不想要的方向]

请：
1. 先给 10 个常规但靠谱的想法
2. 再给 5 个疯狂但可能有趣的想法
3. 最后给 3 个混搭想法（把两个不相关的东西结合起来）

每个想法用一两句话描述即可，不用展开。
```

---

## 使用示例

```
我想为"一个编程学习网站"brainstorm 一些 slogan。

背景信息：
- 目标受众：想学编程但不知道从哪开始的职场人
- 风格偏好：亲切、不装、有行动力
- 不要：太技术的、太鸡汤的

请：
1. 先给 10 个常规但靠谱的想法
2. 再给 5 个疯狂但可能有趣的想法
3. 最后给 3 个混搭想法
```

---

## 创意激发技巧

| 方法 | 说明 | Prompt 补充 |
|------|------|-------------|
| 反向思维 | 从"不要什么"出发 | "先列出 10 个最烂的方案，再反转它们" |
| 类比迁移 | 把其他领域的方法借鉴过来 | "如果 [其他行业] 来做这件事，会怎么做？" |
| 极端假设 | 假设条件极端化 | "如果预算无限 / 只有一天 / 用户是外星人..." |
| 组合创新 | 把两个不相关的东西结合 | "把 A 和 B 结合起来会怎样？" |

---

## 技巧

- **分三轮生成**：常规 → 疯狂 → 混搭，逐步突破思维定式
- **给出约束**："不要什么"和"要什么"同样重要
- **挑 3 个深入**：从 18 个想法中挑 3 个最好的，让 AI 展开
- **迭代优化**：把初步想法给 AI，让它进一步打磨
