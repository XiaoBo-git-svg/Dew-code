# 故事创作

> AI 能帮你快速生成故事大纲、人物设定、甚至完整的初稿。

---

## Prompt 模板

```
我想写一个 [故事类型] 的故事，请帮我构思。

类型：[比如：科幻 / 悬疑 / 爱情 / 职场]
长度：[比如：短篇小说 / 长篇小说 / 电影剧本]
核心设定：[比如：一个能看到别人剩余寿命的人 / 一家专门帮人实现愿望的店]
主角：[主角的基本情况，可简单可详细]
想表达的主题：[比如：选择的重要性 / 人与 AI 的关系]

请帮我生成：
1. 一个有起承转合的故事大纲
2. 3-5 个主要人物的设定（性格、背景、在故事中的作用）
3. 3 个可能的结局（喜剧、悲剧、开放结局）
4. 几个关键的冲突点
```

---

## 创作流程

| 步骤 | 内容 | Prompt 要点 |
|------|------|-------------|
| 1. 构思 | 故事大纲、人物设定 | 给 AI 充分的背景信息 |
| 2. 大纲 | 章节结构、情节走向 | 要求起承转合 |
| 3. 初稿 | 分章节撰写 | 每次写一章，保持上下文 |
| 4. 润色 | 风格统一、细节丰富 | 指定文学风格 |
| 5. 修改 | 根据反馈调整 | 明确修改方向 |

## 人物设定模板

```
请帮我设计一个故事人物。

角色定位：[主角 / 反派 / 配角]
故事背景：[故事发生的世界/时代]
角色功能：[在故事中起什么作用]

请给出：
1. 姓名和外貌特征
2. 性格特点（至少 3 个优点和 2 个缺点）
3. 背景故事（为什么会成为现在这样）
4. 口头禅或标志性动作
5. 与其他角色的关系
```

---

## 使用示例

```
我想写一个科幻悬疑的故事，请帮我构思。

类型：科幻悬疑
长度：短篇小说
核心设定：2045 年，人类可以将记忆上传到云端，但有人发现云端记忆被篡改了
主角：一位网络安全工程师，30 岁，性格内向但观察力极强
想表达的主题：记忆与身份的关系——如果记忆是假的，你还是你吗

请帮我生成：
1. 一个有起承转合的故事大纲
2. 3-5 个主要人物的设定（性格、背景、在故事中的作用）
3. 3 个可能的结局（喜剧、悲剧、开放结局）
4. 几个关键的冲突点
```

---

## 技巧

- **先构思再写**：让 AI 先生成大纲和人物，确认后再写正文
- **指定风格**：想要鲁迅风格还是东野圭吾风格，差别很大
- **分章节写**：长篇小说一次写一章，保持上下文连贯
- **多给约束**：越具体，AI 的输出越符合你的期望
- **保留修改权**：AI 的初稿是起点，你的修改才是灵魂
