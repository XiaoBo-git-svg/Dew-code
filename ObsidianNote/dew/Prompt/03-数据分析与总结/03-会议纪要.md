# 会议纪要

> 开会一小时，整理纪要两小时——AI 能帮你把这个时间缩短到 10 分钟。

---

## Prompt 模板

```
请帮我把以下会议内容整理成一份清晰的会议纪要。

会议基本信息：
- 会议名称：[比如：项目周会]
- 时间：[日期]
- 参会人：[名单]

要求格式：
1. 会议概述（一句话总结这次会开了什么）
2. 讨论要点（每个点用简短的话总结）
3. 行动项（表格形式：任务 | 负责人 | 截止时间）
4. 待决定事项（还没结论的问题）
5. 下次会议安排

会议内容：
[录音转写的文字 / 你的笔记]
```

---

## 使用示例

```
# 会议纪要：项目第 5 次周会

## 会议概述
本次会议回顾了各模块进度，讨论了 3 个待解决问题，确定了下周演示计划。

## 讨论要点
- 用户模块：登录/注册/密码重置完成，短信接口有超时问题
- 支付模块：微信/支付宝测试环境已通，生产密钥待审批
- 商品模块：图片上传慢，需优化压缩或分片上传
- 前端：用户中心/商品列表/购物车完成，下周做订单页面

## 行动项

| 任务 | 负责人 | 截止时间 |
|------|--------|----------|
| 跟进短信接口问题 | 张三 | 每天同步 |
| 联系财务催促支付密钥 | 李四 | 本周五 |
| 调研图片压缩方案 | 王五 | 周四前 |
| 准备阶段性演示 | 全员 | 下周三 |

## 待决定事项
- 图片压缩方案待王五调研后决定

## 下次会议
下周五下午 2 点，302 会议室
```

---

## 技巧

- **录音转文字**：用讯飞听见、飞书妙记、Whisper 等工具先转写
- **输入原始笔记**：不需要整理得很整齐，AI 能理解零散的笔记
- **强调行动项**：这是会议纪要最有价值的部分
- **补充参会人信息**：帮助 AI 正确归因任务
