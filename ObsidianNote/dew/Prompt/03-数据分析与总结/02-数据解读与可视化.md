# 数据解读与可视化

> 你不需要成为数据专家，也能从数据里看出东西。

---

## Prompt 模板

### 数据分析

```
这是一份 [数据描述]，请帮我分析一下：

1. 数据里有什么明显的趋势？
2. 有什么异常值吗？
3. 有什么值得注意的发现？
4. 你建议我深入分析哪些方面？

数据：
[你的数据]
```

### 图表选择

```
我有一组 [数据类型]，想要展示 [想说明的问题]。

应该用什么类型的图表？请给出 2-3 个选项，说明各自的优缺点。
```

### SQL 生成

```
我有一张表叫 [表名]，字段有：
- [字段1]：[类型]，[说明]
- [字段2]：[类型]，[说明]
- ...

我想查询 [查询需求]，请帮我写 SQL。
```

### Excel 公式

```
在 Excel 中，我想要 [需求描述]。

数据位置：
- [数据1] 在 [单元格位置]
- [数据2] 在 [单元格位置]

应该用什么公式？请给出完整公式和简要说明。
```

---

## 使用示例

```
这是一份电商网站近 6 个月的月度销售数据，请帮我分析一下：

1. 数据里有什么明显的趋势？
2. 有什么异常值吗？
3. 有什么值得注意的发现？
4. 你建议我深入分析哪些方面？

数据：
| 月份 | 订单量 | 客单价 | 退款率 |
|------|--------|--------|--------|
| 1月  | 1200   | 89元   | 3.2%   |
| 2月  | 980    | 92元   | 2.8%   |
| 3月  | 1350   | 85元   | 4.1%   |
| 4月  | 1420   | 88元   | 3.5%   |
| 5月  | 2100   | 76元   | 5.2%   |
| 6月  | 1800   | 91元   | 3.0%   |
```

---

## 技巧

- **提供数据上下文**：告诉 AI 数据的来源、时间范围、业务背景
- **先问趋势再问原因**：先让 AI 发现现象，再分析原因
- **要求可视化建议**：让 AI 推荐图表类型，你再用工具实现
