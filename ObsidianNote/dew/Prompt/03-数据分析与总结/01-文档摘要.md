# 文档摘要

> 这是最实用的功能之一——把长文档、长视频、长录音扔给 AI，让它帮你提取关键信息。

---

## Prompt 模板

```
请帮我处理以下内容，生成一份摘要。

内容类型：会议记录 / 文章 / 视频字幕 / 邮件对话 / 培训材料

要求：
1. 先给一个 100 字以内的总览
2. 列出 3-7 个核心要点
3. 提取所有的行动项（谁、什么时候、做什么）
4. 列出关键数据和重要结论
5. 如果有待决定的问题，单独列出来

内容：
[你的内容]
```

---

## 使用示例

```
请帮我处理以下内容，生成一份摘要。

内容类型：会议记录

要求：
1. 先给一个 100 字以内的总览
2. 列出 3-7 个核心要点
3. 提取所有的行动项（谁、什么时候、做什么）
4. 列出关键数据和重要结论
5. 如果有待决定的问题，单独列出来

内容：
张总：今天主要讨论 Q3 的营收目标，目前完成 78%，还差 220 万。市场部说线下活动的效果比线上好，建议加大线下投入。
李经理：同意，我们做过统计，线下获客成本是线上的 60%，转化率高 40%。
王总监：技术这边说新版 APP 9 月 15 号能上线，需要测试团队提前安排人手。
张总：好，测试安排 3 个人，9 月 1 号开始。Q3 剩余目标拆到每周，下周一各部门提交计划。
```

---

## 不同内容类型的使用

| 内容类型 | 预处理方式 | 注意事项 |
|----------|------------|----------|
| 会议记录 | 直接输入 | 区分发言人 |
| 文章/报告 | 直接输入 | 长文章分段处理 |
| 视频/音频 | 先用 Whisper 转文字 | 转写可能有错误，需要校对 |
| 邮件对话 | 按时间顺序整理 | 标注发件人 |
| PDF 文档 | 提取文字后输入 | 注意格式丢失 |

---

## 技巧

- **指定摘要格式**：bullet point 还是段落，根据需求选择
- **要求提取行动项**：这是最有价值的部分
- **视频/音频先转文字**：用 Whisper 等工具，再让 AI 处理
- **分段处理长文档**：超过 5000 字建议分段
